Vanilla Scroll Sky

A pure CSS scrollytelling utility for sticky image reveals and moving captions. No JavaScript, no Web Components, no framework.

This demo uses normal HTML, CSS classes, CSS Custom Properties, @layer, @scope, and a named container.

Test 1: image reveal from left

The image uses .vss--image-left. The caption has no motion class, so it scrolls vertically in the normal document flow.

Full pass-through captions

The next cases are stronger visual effects. They are useful for short labels, but less ideal for long reading text.

Empty caption test

The next instance omits the caption element. This is the recommended way to create an image-only story section.

Fallback behavior

If scroll-driven animations are not supported, the image and caption stay visible and the sticky layout still works. If the user prefers reduced motion, transform animations are removed.
